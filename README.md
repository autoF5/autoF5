# autoF5 (웹사이트 다중접속기)

따로 다운로드나 세팅이 필요 없이 브라우저상에서    
바로 웹사이트를 다중으로 접속하여   
성능이나 기타문제를 체크해볼 수 있는 용도입니다  

웹사이트 성능테스트 용도로만 사용하시기를 권고드립니다
